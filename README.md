# Design Patterns Lab

This project demonstrates various design patterns in Java 17.

## Creational Patterns

- Singleton: Run `java -cp creational/build/libs/creational.jar com.example.creational.singleton.Demo`
- Factory Method: Run `java -cp creational/build/libs/creational.jar com.example.creational.factorymethod.Demo`
- Builder: Run `java -cp creational/build/libs/creational.jar com.example.creational.builder.Demo`
- Prototype: Run `java -cp creational/build/libs/creational.jar com.example.creational.prototype.Demo`
- Abstract Factory: Run `java -cp creational/build/libs/creational.jar com.example.creational.abstractfactory.Demo`

## Behavioral Patterns

- Observer: Run `java -cp behavioral/build/libs/behavioral.jar com.example.behavioral.observer.Demo`
- Strategy: Run `java -cp behavioral/build/libs/behavioral.jar com.example.behavioral.strategy.Demo`
- Command: Run `java -cp behavioral/build/libs/behavioral.jar com.example.behavioral.command.Demo`
- Iterator: Run `java -cp behavioral/build/libs/behavioral.jar com.example.behavioral.iterator.Demo`
- Template Method: Run `java -cp behavioral/build/libs/behavioral.jar com.example.behavioral.templatemethod.Demo`

## Structural Patterns

- Adapter: Run `java -cp structural/build/libs/structural.jar com.example.structural.adapter.Demo`
- Decorator: Run `java -cp structural/build/libs/structural.jar com.example.structural.decorator.Demo`
- Facade: Run `java -cp structural/build/libs/structural.jar com.example.structural.facade.Demo`
- Proxy: Run `java -cp structural/build/libs/structural.jar com.example.structural.proxy.Demo`
- Composite: Run `java -cp structural/build/libs/structural.jar com.example.structural.composite.Demo`

To build: `./gradlew build`

To run a specific demo, first build, then use the java command as above.
